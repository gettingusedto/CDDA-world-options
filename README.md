# CDDA-world-options
Set of world options to modify in your world as a mod.

The mod starts out with the default numbers so you may want to change the numbers to suit your needs.

# Working version
Guesstimating at experimental and stable builds that's beyond [#79310](https://github.com/CleverRaven/Cataclysm-DDA/pull/79310). Might not work prior.

# Warning
Tweaking with the world settings are unstable and may cause glitches and bugs that can break your saves. **It also breaks the warranty**. By using the mod, you acknowledge the fact that it can break your save.
